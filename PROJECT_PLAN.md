# План проекта: Qazaq Carbon Academy (Интерактивный портал)

**Цель:** Создать прозрачную информационную экосистему для фермеров, объясняющую сложные международные методики углеродной сертификации (Verra, Gold Standard и др.) на понятном языке.

## Текущий статус: Этап 1 (Создание статических макетов / Frontend прототипов)

### Сделано:
- [x] Разработан фирменный дизайн-код (UI/UX) с использованием цветов Qazaq Carbon (черный, синий, желтый) и шрифтов (Outfit, Inter).
- [x] Создана Главная страница (`index.html`) с блоками доверия и дорожной картой проекта.
- [x] Интегрирован и стилизован Углеродный Калькулятор (`calculator.html`, `calc.js`, `calc.css`) с учетом климатических зон и формул IPCC (2006). Добавлены новые породы деревьев (саксаул, вяз, осина, лиственница, кедр).
- [x] Создан прототип раздела "Справочник стандартов" (`standards.html`) с карточками для загрузки оригинальных PDF-документов и перехода к простым разборам.

### В процессе:

## Будущие шаги (Архитектура и Разработка)

### Этап 2: Сборка интерактивного "Движка" (Frontend)
- Перенос статических HTML/CSS макетов на современный фреймворк (например, **React / Next.js**). Это обеспечит мгновенную загрузку страниц, плавные переходы и улучшенную SEO-оптимизацию.
- Разделение интерфейса на переиспользуемые компоненты (кнопки, карточки, секции).

### Этап 3: Подключение Базы Данных (CMS - Система управления контентом)
- Регистрация и настройка облачной Headless CMS (например, **Sanity** или **Strapi**).
- Создание структуры данных (Схемы) для стандартов. Поля будут включать: Название, Тип (ARR, SOC и т.д.), Оригинальный PDF-файл, Текст простого разбора, Изображения/Схемы.
- *Действие клиента:* Начать перенос текстов и загрузку PDF-файлов через удобный визуальный интерфейс панели администратора.

### Этап 4: Интеграция Frontend и Backend
- Написание программного кода, который будет автоматически запрашивать данные из системы управления контентом (CMS) и отображать их на страницах стандартов.
- Тестирование на мобильных устройствах и планшетах (Responsive Design).
- Проверка правильности работы калькулятора в новой среде.

### Этап 5: Запуск в Интернет (Деплой)
- Размещение готового проекта на быстром облачном хостинге (например, **Vercel**).
- Привязка существующего домена (`www.qazaqcarbon.com` или поддомена, например, `academy.qazaqcarbon.com`).
- Настройка SSL-сертификатов для безопасного соединения (HTTPS).

## Требования к клиенту к этапу запуска:
1. Подготовленные тексты (простые разборы) для первых стандартов (например, Verra VM0047).
2. Оригинальные PDF-документы стандартов.
3. Доступы к панели управления регистратора домена (`qazaqcarbon.com`) для настройки DNS-записей.
